📋 How to prepare your CSV file

Your file needs these columns — one row per student. Column order doesn't matter.

👤
student_name
Student's full name for display
Optional — defaults to "Student 1, 2…"
📚
study_hours_per_week
How many hours the student studies per week
Range: 1 – 40
🏫
attendance_rate
Fraction of classes attended
Range: 0.0 – 1.0  (e.g. 0.85 = 85%)
📊
previous_gpa
GPA from the previous term
Range: 0.0 – 4.0
✅
assignments_completed
Percentage of assignments submitted
Range: 50 – 100
🧑‍🏫
tutoring_sessions
Number of tutoring sessions attended
Range: 0 – 20
😴
sleep_hours
Average hours of sleep per night
Range: 4 – 10
💼
part_time_job
Does the student have a part-time job?
0 = No  |  1 = Yes
🎭
extracurriculars
Number of extracurricular activities
Range: 0 – 8
🏠
socioeconomic_index
Background socioeconomic score
Range: 1 – 10

Example row

Alice Njeri, 18, 0.92, 3.4, 90, 4, 7.5, 0, 2, 8
